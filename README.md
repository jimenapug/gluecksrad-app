# Glücksrad mit Abstimmungsfaktor

Dieses Projekt ist eine Webanwendung, die ein Abstimmungssystem mit einem Glücksrad kombiniert. Nutzer können eine eigene Abstimmungsfrage mit mehreren Antwortmöglichkeiten erstellen. Die Ergebnisse der Abstimmung werden anschließend in einem Glücksrad visualisiert, wobei die Größe der einzelnen Segmente proportional zur Anzahl der abgegebenen Stimmen ist.

## Funktionen

* Erstellung einer individuellen Abstimmungsfrage
* Hinzufügen beliebig vieler Antwortoptionen
* Stimmabgabe für die verschiedenen Optionen
* Dynamische Anpassung des Glücksrads anhand der Stimmenverteilung
* Zufällige Auswahl eines Ergebnisses durch eine Spin-Animation
* Einfache und intuitive Benutzeroberfläche
* Responsives Webdesign

### Sprachanteile

* JavaScript: **68,3 %**
* CSS: **22,5 %**
* HTML: **9,2 %**

## Nutzung

1. Eine Abstimmungsfrage eingeben.
2. Mindestens zwei Antwortoptionen hinzufügen.
3. Die Umfrage speichern.
4. Stimmen für die gewünschten Optionen abgeben.
5. Auf **„Drehen“** klicken, um das Glücksrad entsprechend der Stimmenverteilung rotieren zu lassen.
